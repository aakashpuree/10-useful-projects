About
// What this tool does

DATE_FINDER is a pure client-side search tool that lets you query 326 banknote records from the ideas.sql database. No server. No page reload. Every record is embedded directly into the JavaScript bundle. Search by full date, year alone, month alone, day alone, or any two-part combination. Unmatched queries can be saved to your local history.
